# static activateTracking(ctx) → {Object}

This is how all tracked properties are converted from the place holder TrackedProperty object into actual tracked getter/setters. Any object that uses tracked must be passed through activateTracking otherwise the properties are useless.

let obj = activateTracking({
  foobar: tracked('FOO')
});

# static defineTrackedProp(ctx, propName, initialValueopt)

Manually add a tracked getter/setter to an object. This can be used in lieu of tracked and activateTracking

let obj = {};
defineTrackedProp(ctx, 'foobar', 'FOO');

Is the same as

let obj = activateTracking({
  foobar: tracked('FOO')
});

# static setOnTagDirtied(callback)

Register a function to be called when a tracked property is dirtied. There can only be one. This is not used when working with the renderer or Component class as they mange this for you.

You would only use this if you were interfacing with this module on your own.
